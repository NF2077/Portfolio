Selamat datang di repositori portofolio utama saya. Ruang digital ini dirancang sebagai platform interaktif untuk menampilkan keahlian teknis, rekayasa kode, serta selera desain saya dalam membangun aplikasi web modern.

Seluruh ekosistem portofolio ini dibangun menggunakan **Pure Vanilla JavaScript (ES6+)**, HTML5 semantik, dan CSS3 modern tanpa bergantung pada *framework* pihak ketiga demi mencapai performa pemuatan yang maksimal (*Zero Dependencies*).

---

## 🛠️ Fitur & Arsitektur Utama

* **🌐 Semantic HTML5:** Struktur kode yang bersih untuk optimasi SEO, ramah mesin pembaca (*Screen Reader*), dan memenuhi standar aksesibilitas web (A11y).
* **🎨 CSS3 Custom Properties (Variables):** Manajemen arsitektur warna terpusat dengan estetika *Dark Mode* berkelas, siap untuk skalabilitas tema di masa mendatang.
* **📱 Ultra-Responsive Component Layout:** Tata letak cair menggunakan CSS Grid dan Flexbox, menjamin tampilan presisi baik di layar *smartphone* maupun monitor *ultrawide*.
* **✨ Vanilla Micro-Interactions:** Implementasi logika JavaScript murni untuk memberikan pengalaman navigasi yang dinamis dan interaktif saat pengguna menjelajahi konten.
